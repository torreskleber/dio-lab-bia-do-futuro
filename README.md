## 🎯 **Visão Geral**

O **Vio** é um agente de IA Generativa projetado para atuar como **educador financeiro digital**, oferecendo explicações claras, análises contextualizadas e orientações seguras com base em dados estruturados.

Este projeto foi desenvolvido como solução prática no laboratório **BIA do Futuro (DIO + Bradesco)**, evoluindo um template inicial para uma solução **completa, funcional e profissional**, ideal para compor um portfólio de destaque.

## 🧠 **Arquitetura do Agente**

O Vio foi construído sobre três pilares principais:

### **1) IA Generativa (LLM)**
- Interpretação de linguagem natural

### **2) Base de Conhecimento Estruturada**
Localizada em `data/`, contendo:

Documentadas em `docs/`, incluindo:

- Persona e tom de voz  
- Estratégias anti-alucinação  
- Limites de atuação  
- Métricas de qualidade  
- Exemplos de interação

# 💰 Vio — Educador Financeiro com IA Generativa

> Agente inteligente e proativo que orienta, educa e personaliza decisões financeiras com base no perfil real do cliente.

---

## 🧠 Sobre o Projeto

O **Vio** é um educador financeiro virtual desenvolvido com IA Generativa, criado como parte do desafio **BIA do Futuro** da [DIO](https://www.dio.me/). Ele vai além de um chatbot reativo: antecipa necessidades, personaliza sugestões e cocria soluções financeiras de forma consultiva e segura.

---

## ✨ Funcionalidades

- 💬 Conversa natural sobre finanças pessoais e investimentos
- 📊 Análise do perfil do investidor e histórico de transações
- 🎯 Sugestões personalizadas de produtos financeiros
- 🛡️ Respostas confiáveis com mecanismos anti-alucinação
- 📚 Tom educativo, acessível e sem jargões desnecessários

## 🚀 Como Executar

### Pré-requisitos

- Python 3.9+
- Uma chave de API OpenAI ou Anthropic (Claude)

### Instalação

```bash
# Clone o repositório
git clone https://github.com/torreskleber/dio-lab-bia-do-futuro.git
cd dio-lab-bia-do-futuro

# Instale as dependências
pip install -r requirements.txt

# Configure sua chave de API
export OPENAI_API_KEY="sua-chave-aqui"
# ou
export ANTHROPIC_API_KEY="sua-chave-aqui"

# Inicie a aplicação
streamlit run src/app.py
```
